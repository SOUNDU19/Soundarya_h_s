# Soundarya H S - Portfolio

A modern, professional portfolio website showcasing my projects, experience, and skills in AI/ML and Web Development.

## Features

- **Glassmorphism Design** - Modern frosted glass UI effects
- **Smooth Animations** - AOS scroll animations and interactive hover effects
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Projects Showcase** - Highlighting AI-powered ticket management system and other projects
- **Experience Section** - Details of AI/ML internship at Infosys Springboard
- **Interactive Elements** - 3D card tilt effects and smooth transitions

## Technologies Used

- HTML5
- CSS3 (Glassmorphism, Animations)
- JavaScript (ES6+)
- AOS (Animate On Scroll Library)
- Font Awesome Icons
- Google Fonts (Poppins)

## Sections

1. **Hero** - Introduction with profile photo and social links
2. **About Me** - Background and interests
3. **Education** - Academic qualifications
4. **Skills** - Technical and soft skills with tab navigation
5. **Projects** - Featured projects with live demos
6. **Experience** - Professional internship experience
7. **Contact** - Get in touch section
